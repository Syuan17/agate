# Agate
Agate is a cloud-native, fast, scalable, and async API Gateway. its core values are high performance and extensibility.

# Dependency
- Vert.x 4.0
- Consul 1.7

# Config
- Proxy: /(?<url>.*)

# Feature
- Logging
- RateLimit
- Tracing
- Metrics
- Security
- CircuitBreaker
- GrayDeployment
- Routing: Http URL Proxy
- Routing: Http Microservice Discovery
